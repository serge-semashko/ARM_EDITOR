Порядок установки

1. InfovoxDesktop2.220Engine - собственно голосовой движок
2. Russian_ID2220.exe - голос Алёны
3. IDSP_203 - патч голосового движка

Используем генератор ключей для регистрации. Файл ключа кладётся на рабочий стол.
Заходим в папку C:\Program Files\Acapela Group\Infovox Desktop 2.2, находим
и запускаем файл BabLicMan.exe. В открывшемся окне жмём кнопку Import License File
и указываем на лежащий на Рабочем столе файл Infovox.lic. Если устаналивать поверх
имеющейся версии 2.2, лицензия сохраняется, можно повторно не лицензировать.
